# 自傳簡介

我是一名前端工程師，擁有 2 年前端實作經驗與 2 年 UI 設計背景。專注於透過軟體前端技術的應用，為中小企業執行品牌的數位內容傳播與網路系統整合。

## 技術專長

我主要使用 React 生態系、TypeScript、Tailwind CSS 等現代 Web 技術進行開發。在服務企業客戶時，我負責前端界面開發、API 串接、基礎微後端部署，以及跨裝置響應式版面的翻新與實作。從視覺設計到程式實作，我能夠完整掌握整個開發流程。

## 工作範疇

我的工作涵蓋多個面向：

**品牌網站設計**：為企業打造視覺導向的 UI 介面，製作具備當代網頁技術的 Web 與 Mobile 互動體驗。曾為上市櫃企業、品牌公司、公關公司等不同產業客戶開發客製化網站。

**互動遊戲開發**：使用 Angular.js 等技術製作跨平台的手機互動小遊戲，為品牌方建立角色 IP，並透過遊戲進行行銷推廣。這些作品不僅是技術的展現，更是品牌故事的延伸。

**系統建置與自動化**：結合 Node.js、Python、n8n、LLM 等工具，開發自動化流程與系統。例如為工廠建置 ERP 系統，將傳統紙本紀錄數位化；為高爾夫球場開發 CRM + POS 系統；為中小企業開發 YouTube 頻道自動化內容生成工具等。

**AI 整合應用**：探索 Generative AI、提示詞工程等新興技術，將 AI 圖片批次自動化生成、企業 AppScript 整合等應用落實在實際專案中。

## 工作特色

我善於快速迭代（Iteration）與落地建構原型（Prototype）的能力。透過大量的 Side Projects，我持續累積不同業態的專案開發經驗，從混音工具、n8n workflow 到各種創意應用，不斷拓展技術邊界。

我相信技術不應該只是工具，而是能夠為品牌創造價值、為使用者帶來美好體驗的媒介。每一行程式碼背後，都承載著對品質的堅持與對創新的追求。
